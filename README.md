<h1 align="center"> Sistema de Permutas 🔄</h1>

<p>Este repositório contém um Sistema de Permutas, desenvolvido para facilitar e organizar trocas de turno entre funcionários de uma empresa, garantindo transparência, rastreabilidade e aprovação adequada das solicitações.

Embora o foco principal seja troca de turnos, o sistema foi pensado para ser flexível e pode ser expandido futuramente para outros tipos de permutas, como trocas de tarefas, escalas, plantões, setores ou até mesmo recursos internos.</p>

<div align="center">

<h2>Objetivos</h2>
  
O objetivo principal do sistema é permitir que funcionários possam:

>Solicitar troca de turno com outro colaborador, onde essas solicitações serão enviadas para aprovação por parte dos supervisores/gestores. O supervisor ou gestor pode aceitar ou recusar solicitações recebidas, manter histórico completo das trocas realizadas. Assim evitando conflitos de horários e garantindo a conformidade das regras internas da empresa.
</div>
